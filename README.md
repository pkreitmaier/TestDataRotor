# Python Solution

## Contents

- `solution.py` – Python script containing the complete solution.
- `plot/` – Figures generated by the script.

## Requirements

Install the required Python packages:

```bash
pip install -r requirements.txt
```

## Input data

The input dataset is not included in this repository and must be provided separately.

Before running the script, update the `DATA_FILE` variable at the beginning of `solution.py` so that it points to the absolute path of `TestDataRotor.csv`.

## Running the script

Run:

```bash
python solution.py
```

The numerical results are printed to the standard output (terminal). For convenience, the corresponding output is also included as comments in `solution.py`.

The generated figures are displayed during execution and are also saved in the `plot/` directory.
